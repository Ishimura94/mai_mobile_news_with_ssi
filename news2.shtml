Windows исполнилось 30 лет

Сегодня, 20 ноября 2015 года, корпорация Microsoft отмечает важную дату в истории операционной системы Windows. Ровно 30 лет назад, 20 ноября 1985 года, состоялся релиз самой первой версии Windows. За 30 лет компания Microsoft выпустила 14 версий ОС, а самой последней из них стала Windows 10. На сегодняшний день самой популярной "операционкой" остаётся Windows 7, которая установлена на 55,71% всех персональных компьютеров в мире. Несмотря на это, число пользователей Windows 10 стремительно растёт, и с момента релиза платформы, состоявшегося 29 июля 2015 года, её пользовательская база превысила барьер в 120 миллионов и продолжает стремительно расти.

Список всех версий Windows за 30 лет существования:

  • Windows 1.0 (1985);
  • Windows 2.0 (1987);
  • Windows 3.x (1990, 1992);
  • Windows NT (1993);
  • Windows 95 (1995);
  • Windows 98 (1998);
  • Windows 2000 (2000);
  • Windows ME (2000);
  • Windows XP (2001);
  • Windows Vista (2006);
  • Windows 7 (2009);
  • Windows 8 (2012);
  • Windows 8.1 (2013);
  • Windows 10 (2015).

Windows 10 стала первой версией Windows, которая ориентирована на все типы устройств. На базе этой платформы работают ноутбуки, настольные компьютеры, игровые консоли, планшеты и смартфоны. К слову, в ближайшее время состоится запуск Windows 10 Mobile, обновление до которой пользователи начнут получать уже в следующем месяце.
